Data Tables
===========
Upon loading in data, the tables can be altered to emphasize trends, adjust the precision of data values, and more generally, make the data easier to interpret. 

.. list-table::
   :widths: 10 30
   :header-rows: 1

   * - Data Table Tool
     - Description
   * - `Loading Data <load.html>`_
     - Opening and importing data files.
   * - `Adjusting Precision <precision.html>`_
     - Adjusts the number of decimal places shown in the tables.
   * - `Filtering Data <filter.html>`_
     - Display desired identifiers and set numerical filters. 
   * - `Pivoting Data <pivot.html>`_
     - Rearranges data columns to emphasize trends. 
   * - `Unpivoting Data <unpivot.html>`_
     - Undoes previous pivotting. 
   * - `Splitting Data <split.html>`_
     - Forms new data tables for rows of the same categorical value. 
   * - `Merge Data <merge.html>`_
     - Combines two data tables based on categorical columns.
   * - `Order Categories <order.html>`_
     - Rearranges order of categorical columns.
   * - `Concat Data <concat.html>`_
     - Combines two data tables.
   * - `Sort <sort.html>`_
     - Reorders rows in ascending order of a selected data feature.
   * - `Saving Data <save.html>`_
     - Saves data table.
